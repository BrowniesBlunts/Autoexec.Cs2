# Autoexec.cfg – Counter-Strike 2

## Descripción

Este repositorio contiene mi archivo `autoexec.cfg` personalizado para **Counter-Strike 2 (CS2)**.  
Su objetivo es aplicar automáticamente configuraciones clave orientadas al **rendimiento**, la **estabilidad** y la **consistencia competitiva** desde el inicio del juego.

El uso de un autoexec permite mantener un entorno de juego controlado, replicable y optimizado sin realizar ajustes manuales en cada sesión.

---

## Objetivo

- Automatizar la carga de configuraciones esenciales.
- Optimizar el rendimiento general del juego.
- Garantizar consistencia en controles y sensibilidad.
- Reducir distracciones visuales.
- Facilitar portabilidad y mantenimiento del entorno competitivo.

---

## Requisitos

- Counter-Strike 2 instalado vía Steam.
- Acceso a los archivos locales del juego.
- Permisos de edición en la carpeta `cfg`.

---

## Instalación

1. Descarga o clona este repositorio.
2. Copia el archivo `autoexec.cfg`.
3. Pégalo en la siguiente ruta:

Steam\steamapps\common\Counter-Strike Global Offensive\game\csgo\cfg\


---

## Parámetros de lanzamiento

Para asegurar la ejecución automática del autoexec al iniciar CS2, agrega el siguiente parámetro en Steam:

+exec autoexec.cfg


---

## Estructura del Autoexec

El archivo está organizado por secciones para facilitar su lectura y mantenimiento:

### Configuración de red
Ajustes relacionados con la sincronización cliente-servidor y estabilidad de conexión para un entorno competitivo consistente.

### Configuración de rendimiento
Parámetros orientados a la estabilidad de FPS, reducción de micro-stuttering y uso eficiente de los recursos del sistema.

### Configuración de control
Incluye sensibilidad, bindings personalizados y comandos rápidos enfocados en precisión y memoria muscular.

### Configuración visual y HUD
Opciones relacionadas con HUD, crosshair y viewmodel, diseñadas para maximizar visibilidad y minimizar distracciones.

---

## Ventajas del uso de un autoexec

- Aplicación automática de configuraciones al iniciar el juego.
- Consistencia total entre sesiones y equipos.
- Entorno competitivo estable.
- Fácil respaldo y versionado mediante Git.
- Ahorro de tiempo en ajustes manuales.

---

## Mantenimiento

Debido a las actualizaciones frecuentes de CS2, se recomienda:

- Verificar compatibilidad tras cada parche.
- Eliminar comandos obsoletos.
- Documentar cambios relevantes dentro del archivo.
- Mantener el repositorio actualizado.

---

## Notas

Las configuraciones incluidas están adaptadas a preferencias personales y pueden variar según hardware, resolución, DPI y estilo de juego.  
Se recomienda ajustar el autoexec según las necesidades individuales.
